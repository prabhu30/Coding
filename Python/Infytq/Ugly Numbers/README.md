# Ugly Numbers

# Question :

[1, 2, 3, 4, 5, 6, 8, 9, 10, 12, 15, â€¦] is the sequence of ugly numbers.

# Task:

# Hint:

The following tc lines contains the number n which you need to check weather it is ugly or not.
